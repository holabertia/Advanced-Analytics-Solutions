4 Entranamiento y Comparativa de modelos

En este apartado entrenaremos y compararemos los modelos mencionados en la explicacin terica de los modelos de clasificacin (Regresin logstica, KNN, SVM, Naive Bayes, rbol de decisin y random forest).

El objetivo es identificar cul de todos ellos se adapta mejor a esta problemtica de clasificacin dividiendo los datos en test y train para ver como se comportan cada uno.

In[9]:
# Extraccin de las variables independientes (X) y la variable dependiente (y)
X = df.drop(['class'], axis=1)
y = df["class"]

# Divisin del dataset en training y test
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=42)
In[10]:
# Listado de modelos a entrenar
models = [LogisticRegression(), KNeighborsClassifier(n_neighbors=10, weights='uniform', algorithm='auto'), SVC(), GaussianNB(), DecisionTreeClassifier(), RandomForestClassifier()]
models_list = ["Logistic Regression", "KNN", "SVM", "Naive Bayes", "Decision Tree", "Random Forest"]
accuracies = []
precisions = []
recalls = []
aucs = []
confusion_matrixs = []
rocs = []
# Entrenamiento de modelos y obtecin de valores de prediccin
for model in models:
    model.fit(X_train, y_train)
    y_pred = model.predict(X_test)
    accuracies.append(accuracy_score(y_test, y_pred))
    precisions.append(precision_score(y_test, y_pred))
    recalls.append(recall_score(y_test, y_pred))
    fpr, tpr, _ = roc_curve(y_test, y_pred)
    rocs.append((fpr, tpr))
    aucs.append(round(auc(fpr, tpr), 2))
    confusion_matrixs.append(confusion_matrix(y_test, y_pred))

    sub = fig.add_subplot(2, 3, i + 1)
    sub.plot(rocs[i][0], rocs[i][1], color='powderblue', lw=2, label='ROC curve (area = %0.2f)' % aucs[i])
    sub.plot([0, 1], [0, 1], color='sandybrown', lw=2, linestyle='--', label='Random Guess')
    sub.set_xlim([0.0, 1.0])
    sub.set_ylim([0.0, 1.05])
    sub.set_xlabel('False Positive Rate')
    sub.set_ylabel('True Positive Rate')
    sub.set_title(model_name)
    sub.legend(loc='lower right')

Tras la obtencin de los resultados finales, podemos ver que la regresin logstica, SVM y Naive Bayes tienen rendimientos muy similares y significativamente bajos, pues al final solo estn acertando un poco ms de la mitad de las veces cuando dan la clasificacin, prcticamente es como si lanzaramos una moneda al aire. El KNN tiene un pequeo mejor resultado en cuanto al accuracy y a la precision, pero no llega a ser competente. Si miarmos las matrices de confusin, vemos que el KNN clasifica ligeramente mejor cuando son comestibles que los otros tres modelos.

Por el otro lado el rbol de decisn y el random forest tienen un rendimiento muy cercano al 100%, siendo ligeramente mejor el random forest. De hecho, si observamos la matriz de confusin podemos ver que el random forest falla la mitad de las veces que el rbol de decisin dando un resultado muy apurado.

Estos resultados indican que para esta estructura de datos lo mejor es algn tipo de rbol. Esto no desmerece al resto de modelos, simplemente que para este problema no se adhieren adecuadamente como si lo pueden hacer en otros.
